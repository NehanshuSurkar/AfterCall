# AfterCall

AfterCall helps users remember what truly matters from phone conversations without taking notes during the call.
Users can record or upload call audio after a conversation, and the app automatically converts speech into text and generates a concise summary with key points and action items. 
Built using dreamflow for rapid, visual app development, AfterCall integrates ElevenLabs to transform summarized content into natural-sounding voice recaps, making it easy to review conversations anytime while keeping user privacy in control.
